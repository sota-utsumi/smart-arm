# smart-arm
## Quickstart

### Prerequisites
- Docker + Docker Compose

### Run (image inference)
1) Put an image at `./input/input.jpg`
2) Run:
   docker compose run --rm arm

### Outputs
- Results: `./runs/`1~## Quickstart
